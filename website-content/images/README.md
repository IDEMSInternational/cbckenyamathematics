# Image Assets

This folder contains all image assets used throughout the website.

## Folder Structure

- **logos/** - Contains logo files for the project and partner organizations
- **screenshots/** - Screenshots of the PreTeXt book, Moodle courses, and other resources
- **icons/** - Icon files used in the UI

## Image Guidelines

- Use optimized images for web (compressed without significant quality loss)
- Recommended formats: PNG for logos/icons, JPG for photos, SVG for scalable graphics
- Include descriptive alt text for all images in HTML
- Name files descriptively (e.g., `pretext-book-screenshot.png`)

## Recommended Sizes

- Logo: Max height 50-60px for header
- Screenshots: Max width 1200px
- Icons: 32x32px or 64x64px for standard use
